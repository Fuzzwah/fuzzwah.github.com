Baby Music Box

Objective: A system to play music to a sleeping baby, which can be controlled remotely and easily triggered.

Solution: Yet another project which I turned to the flexible Raspberry Pi as a base.

On github I found a python and javascript based project called web-mote which had been made to serve a similar purpose, though it was to be run on a raspberry pi which had a screen connected and would play either music or video via mplayer. The web page which provided the remote control had been formatted with an ipod touch in mind.

Working with this as a base I modified the css style sheet to provide a good interface for use on my Android phone. I then realized that while this allow me to trigger playback of the music while still in bed, getting the music playing when we first put our baby down to sleep was more of a hassle than previous, when we just hit play on the mp3 player.

Initially I rigged up a web cam and used the motion software to allow us to trigger playback of music by waving a hand in front of the web cam. However this had one major flaw; music play back would trigger when the room was dark and we turned on the light.

The better solution ended up being a wireless mouse and a python script which would listen for input from the mouse buttons and wheel. Left click would trigger the playback of the mp3s in a folder. Right click stops playback. Mouse wheel up and down changes the volume.

Hi, I'm Fuzzy

Welcome to Fuzzy's Projects.

This tiny little corner of the internet is where I share information about the various projects I've spent time working on. I hope you find it interesting and helpful.

My projects include websites, media players, chrome extensions, automated systems, irc bots, XBMC plugins and online poker related tools. Basically, if I'm interested in something I end up creating something to help me enjoy it more.

I do not claim to be some kind of geek genius, most of my projects are neither original nor unique. I like to believe that I have the spirit of a bricoleur; I create things using whatever materials and information are available. Quite often that means basing my projects on code, tools and hardware created by other people.


fuzzyslogic.com

Save to Website

An extension for Google Chrome which adds a "save to website" option to the menu when you right click on images on web pages.

Along with this extension you'll also need your own webserver and enough nerd points to be able to save and configure a php script that this extension interfaces with. Its super simple, but you'd better know where your web accessible paths are.

Save to Website

Hacker News #1's

HackerNews is a website run by Paul Graham, the co-founding the Y Combinator seed capital firm. Users of the site submit links to news and articles which are of interest to programmers, start up founders and geeks in general. Entries are voted on by other users and the discussion threads which develop are often very insightful.

I’ve created a system which collects the details of links which reach the number 1 position on the site. I use this as a source of interesting reading and it allows me to keep up to date with the hot news even if I’m away from my computer for a few days or otherwise unable to keep up to date with HackerNews.

Home Theater PC

However when we were looking at new TVs we found that we could save around $150 by not getting a new fangled “smart tv”. I figured I could set up a much better system than what the current line up of smart tvs provided for much cheaper. After a little research I found that some extremely smart guys had made this process extremely simple by creating a specially crafted distribution of Debian linux with the sole goal of running XBMC on an R-Pi.

Home Theater PC

RFID Triggered Music Playback

While I’m very pleased with the Raspberry Pi + XBMC system I have in place to watch videos I found myself wanting to implement a different system for music playback.

I came across Swisher, which allows for music playback to be triggered via an RFID near field card reader. The user interacts with the system by touching an RFID card to the reader. Each card is associated with a particular music track in the system and has an image printed on the card to allow the user to visually discern between them.

RFID Triggered Music

XBMC Download Manager

One issue with using the Raspberry Pi to host the XBMC media player is that it lacks the processing grunt to be able to play back high definition video while it runs background services such as Transmission and Sick Beard. I created a small plugin for XBMC which automatically throttles downloads in Transmission and stops the Sick Beard service.

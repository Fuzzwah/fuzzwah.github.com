Home Theater PC

Objective: Provide a full featured home theater media player which is reliable, easy to use and cheap.

Solution: It was a pretty easy choice to use a Raspberry Pi for this task. With a number of teams producing very solid options for running either XBMC or OpenElec on this $35 mini computer. After checking out all of the available choices I decided to base my work on the wonderful XBian solution.

Of course the $35 R-Pi isn't the only piece of hardware required, I've included the full list of items which make up my system. I've also created an Amazon Listmania page for Raspberry Pi XBMC Media Player items.

We'll get into the set up process below the parts.

Back in 2011 when I intitally rigged up my R-Pi XBMC system it was a bit of an effort. The good news for you is, the process is amazingly simple now!

Raspberry Pi Model B

The Raspberry Pi is a credit-card-sized single-board computer developed in the UK by the Raspberry Pi Foundation with the intention of promoting the teaching of basic computer science in schools. The Raspberry Pi has a Broadcom BCM2835 system on a chip (SoC), which includes a 700 MHz processor, VideoCore IV GPU, and 512 MB of RAM. It uses an SD card for booting and persistent storage.

Multicomp Clear Raspberry Pi Enclosure

A solid, simple and cheap case for the Raspberry Pi.

Edimax EW-7811Un 150 Mbps Wireless 11n Nano Size USB Adapter

A cheap and reliable wifi adapter which has excellent driver support in the flavors of Linux we will be using in our various R-Pi projects.

Transcend 16 GB High Speed 10 UHS Flash Memory Card

This 16 gigabyte SD card will be plenty big enough to house our operating system while still having plenty of room for other project related files.

Motorola USB Wall Charger with Micro USB Data Cable

The power source for the Raspberry Pi.

Toshiba 3TB Canvio Desk Desktop External Hard Drive

You're going to need somewhere to store your media files. This 3 terabyte drive provides plenty of space and a great low price.

MCE PC Remote Control and Infrared Receiver

Bargain priced remote control for your media center.

7-port USB3.0 Hub

There are slightly cheaper powered USB hubs, but the fact that this device has switches for each port as well as four USB3 ports can help when transferring data onto the hard drive.

Install Process

Download the latest XBian release from over here. The install process is very simple, as explained in their getting started guide.

Once you've got the XBian files onto your SD card plug it into the R-Pi and get everything else hooked up. The first time you power it on you'll be guided through getting everything rigged up.
